En esta primera version solo coloco la funcion main() en el 
archivo main.c para hacer el commit inicial y subir los 
cambios a github.

$git clone https://upiitacodelamberto/conteosdinamicosyrapidos.git

Para actualizar en servidor de github:
$git add --all

$git commit -m "(2015.11.17) Some comments"

$git push origin master

Para actualizar copia de repositorio local
$git pull origin master
